Usbfmtx

Linux program to configure/control a USB FM transmitter dongle i.e Keene USB FM Transmitter

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Usbfmtx

I bought a USB FM Transmitter on ebay direct from China and I wanted to control it in an easy way. Windows software is available from the Keene Electronics site for the dongle they sell which is identical to the one I bought. I don't use Windows and I couldn't find a Linux program so I decided to write one.

I don't have a lot of experience writing C or C++ software and even less using QT but I decided to learn on the job. I am sure my code could be improved but I wasn't looking to optimise it which, in my opinion, makes it difficult to read and understand.
